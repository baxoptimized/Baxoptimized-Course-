## 📋 Choose your brief

### Brief A — Thirroul Beachside Café

> **Sea & Sand Café** — Thirroul, NSW. Instagram following, no website. Wants foot traffic, menu, beachside vibe, hours. Has nice phone photos. Wants warm, casual, not corporate. **No AHPRA restrictions.**

### Brief B — Illawarra Physio Clinic

> **Illawarra Movement Physiotherapy** — Wollongong. Outdated unprofessional site. Wants something reflecting real expertise, easy booking. Professional headshots only. Wants clean, calm, trustworthy.
> ⚠️ **AHPRA restrictions apply.** Use the AHPRA-safe copywriting prompt (Module 6). No clinical outcome testimonials, no comparative claims, no guarantees.

### Brief C — Local Electrician

> **Sparkwise Electrical** — Shellharbour. Word-of-mouth only, wants emergency call-out and residential search visibility. Phone photos, no logo. Wants fast-loading, trustworthy, very clear emergency contact info. **No AHPRA restrictions.**

> ⚠️ **If you choose Brief B, the AHPRA rules are not optional.** This is a real compliance issue, not a style preference — treat it seriously even as a practice brief.

## ✅ The capstone rubric

*(Same kind of checklist used throughout the course — nothing new to learn.)*

**Design & Build**
- [ ] All required pages built (min: Home, Services/Menu, About, Contact)
- [ ] Mobile-responsive — checked on a narrow viewport
- [ ] Follows Module 5's design fundamentals
- [ ] No placeholder text anywhere
- [ ] Correct voice — AHPRA-compliant if Brief B chosen

**Functionality**
- [ ] Contact form wired to Resend — **real test email sent and received**
- [ ] Phone number clickable on mobile
- [ ] All links work

**Technical & Deployment**
- [ ] Pushed to a **private** GitHub repo, clean commit history
- [ ] Deployed and live on Vercel
- [ ] Environment variables correctly set in Vercel

**SEO & Speed**
- [ ] Title tags + meta descriptions, every page, within limits
- [ ] Image alt text throughout
- [ ] PageSpeed Insights mobile score **90+**
- [ ] GA4 installed and tracking
- [ ] Search Console verified, sitemap submitted

**Visual polish (optional but recommended — Module 7.5)**
- [ ] At least 2-3 visual effects added, matched to the brand's tone

**Going all the way (optional)**
- [ ] Custom domain connected, DNS configured
- [ ] SSL confirmed working
